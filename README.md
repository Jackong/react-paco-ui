# react-paco-ui

React Components for PACO-UI

## UI预览

![qrcode](http://hcz.pingan.com/common/images/paco-ui.png?v=2)

## 使用说明

- 引入CSS

```bash
npm i --save paco-ui
```

- 引入`react-paco-ui`库

```bash
npm i --save react-paco-ui
```

- 代码使用

```js
import 'paco-ui';
import Button from 'react-paco-ui/components/button';
```

> 注：`VERSION`为要使用版本
