# react-paco-ui

React Components for PACO-UI

## UI预览

[UI预览](https://jackong.github.io/react-paco-ui)

## 使用说明

- 引入[paco-ui](https://github.com/Jackong/paco-ui)

```bash
npm i --save paco-ui
```

- 引入`react-paco-ui`库

```bash
npm i --save react-paco-ui
```

- 代码使用

```js
import Button from 'react-paco-ui/components/button';
```
